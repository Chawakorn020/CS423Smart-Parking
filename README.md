#  CS423 Smart Parking System
ระบบลานจอดรถอัจฉริยะ (จำลองการทำงานด้วย Wokwi) พัฒนาเพื่อเป็นโครงงานรายวิชา CS423 ระบบนี้ทำงานแบบอัตโนมัติเต็มรูปแบบ โดยใช้เซนเซอร์ในการตรวจจับและบันทึกเวลาจอด

## 🛠️ เทคโนโลยีที่ใช้ (Platform Stack)
- **Hardware Simulation:** จำลองบอร์ด ESP32, Ultrasonic Sensor (HC-SR04) และ LED (ผ่านแพลตฟอร์ม Wokwi)
- **IoT Protocol:** สื่อสารข้อมูลผ่าน MQTT (HiveMQ Broker) แบบ 1 ทาง (Sensor -> Platform)
- **Middleware/Backend:** ประมวลผลข้อมูล คำนวณเวลาจอด และสร้าง API ด้วย Node-RED
- **Frontend Dashboard:** สร้างหน้าเว็บแสดงผลสถิติและอนิเมชันแบบ Real-time ด้วย Next.js (React) และ Tailwind CSS

## 📦 วิธีการติดตั้งและรันโปรเจกต์ (Installation & Usage)

ในการเปิดใช้งานระบบเต็มรูปแบบ จะต้องรันโปรแกรม 3 ส่วนประกอบกัน ดังนี้:

### 1. รันฝั่ง Hardware (Wokwi)
1. เข้าไปที่โปรเจกต์ Wokwi ตามลิงก์นี้: `[[ให้คุณเอาลิงก์หน้าเว็บ Wokwi ของคุณมาแปะแทนที่ข้อความนี้]](https://wokwi.com/projects/305569599398609473)`
2. กดปุ่ม **Play (▶️)** เพื่อเริ่มการจำลองระบบ
3. ระบบจะทำการเชื่อมต่อ Wi-Fi และเชื่อมต่อเข้ากับ MQTT Broker อัตโนมัติ เซนเซอร์จะเริ่มวัดระยะและส่งข้อมูลแบบ Real-time (ทุก 2 วินาที)

### 2. รันฝั่ง Backend (Node-RED)
1. เปิด Command Prompt (cmd) ขึ้นมา
2. พิมพ์คำสั่งรันระบบ: `node-red` แล้วกด Enter
3. เปิดเว็บเบราว์เซอร์ไปที่ `http://localhost:1880`
4. ทำการ Import ไฟล์ `flows.json` (ที่แนบมาใน Repository นี้) เข้าไปในระบบ
5. กดปุ่ม **Deploy** สีแดงที่มุมขวาบน ระบบจะเริ่มรับข้อมูลจาก Wokwi ทันที

### 3. รันฝั่งหน้าเว็บ Dashboard (Next.js)
1. เปิด Command Prompt (cmd) อีกหน้าต่างหนึ่ง
2. พิมพ์คำสั่งเพื่อเข้าไปในโฟลเดอร์โปรเจกต์ (เช่น `cd smart-parking`)
3. ติดตั้งแพ็กเกจด้วยคำสั่ง: `npm install`
4. สตาร์ทหน้าเว็บด้วยคำสั่ง: `npm run dev`
5. เปิดเว็บเบราว์เซอร์ไปที่ `http://localhost:3000` เพื่อดูผลลัพธ์
6. *หมายเหตุ:* หน้าเว็บมีปุ่ม **"ล้างข้อมูล & รีเซ็ตระบบ"** สำหรับเคลียร์สถิติการจอดที่เก็บอยู่ในหน่วยความจำของ Node-RED ให้กลับเป็นศูนย์
